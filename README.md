# ClickAnimDemo
点击动画效果，无需修改控件和布局
# 基于popupWindow实现</br>
使用非常简单，可以自定义任何效果，封装在基类可以保证所有界面都能有效果</br>
主要也就是在activity中捕获用户点击事件并记录点击位置，
然后在这个位置弹窗并播放动画给人的感觉就像是点击动画一样，
因为确实很简单就不单独写成lirbary了，大家看看代码就懂了，非常easy</br>
